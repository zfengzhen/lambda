# Lambda 策略：一套基于技术面的动态卖 Put 决策框架

## 核心思想

Lambda 策略是一个系统化的**裸卖看跌期权（Sell Put）决策树**。

它不回答"要不要卖 Put"——答案永远是卖。它回答的是：**卖多深的虚值？**

市场越强，行权价越靠近现价，吃更多权利金；市场越弱，行权价越远离现价，换更大的安全垫。整套逻辑只用日 K 线数据驱动，不依赖主观判断。

---

## 决策树

```
IF   MACD收窄 AND Close > P5_PP AND Close > P30_PP
     → A  企稳双撑     卖 8% OTM Put（行权价 = Close × 0.92）

ELIF Close < MA20 AND Close > MA60
     → B1 回调均线     卖 8% OTM Put（行权价 = Close × 0.92）

ELIF 波动率 < 50% AND |MA20距| ≤ 4.5%
     → B4 低波整理     卖 15% OTM Put（行权价 = Close × 0.85）

ELIF DIF < 0 AND Close > P30_PP
     → B2 超跌支撑     卖 8% OTM Put（行权价 = Close × 0.92）

ELIF MA20 > MA60 AND DIF < 0
     → B3 趋势动能弱   卖 10% OTM Put（行权价 = Close × 0.90）

ELIF Close ≥ MA20 AND |MA20偏离| ≤ 10%
     → C2 趋势延续     卖 15% OTM Put（行权价 = Close × 0.85）

ELIF Close ≥ MA20 AND |MA20偏离| > 10%
     → C3 过热追涨     卖 15% OTM Put（行权价 = Close × 0.85）

ELIF Close < MA60 AND |MACD今| < |MACD昨|
     → C1 跌势减速     卖 10% OTM Put（行权价 = Close × 0.90）

ELSE
     → C4 加速下杀     卖 15% OTM Put（行权价 = Close × 0.85）
```

优先级自上而下，命中即停。

## 连续弱势熔断规则

当市场连续处于弱势（C 类）时，策略会触发熔断暂停卖出，避免在持续下跌中反复被行权。

**触发条件**：前 2 周都是 C 类 + 本周也是 C 类时进入判断（本周是 A/B 类始终放行）：

- **本周 C 类但非 C1** → 暂停。趋势没有减速迹象（C2 还在涨但不稳、C3 过热、C4 加速下杀），风险未收敛。
- **本周 C1 + 前 2 周有 C1** → 继续卖出。跌势已连续收窄（MACD 收窄出现过至少 2 次），空头力量持续衰减，底部信号较可靠。
- **本周 C1 + 前 2 周无 C1** → 暂停。之前都是 C2/C3/C4 纯弱势，本周才首次出现减速，可能只是下跌中继的短暂喘息而非真正见底，不够安全。

**核心思路**：单次 C1 减速不可信，连续减速才可信；A/B 类已有支撑条件，不受熔断影响。

## 技术指标定义

### MACD

采用标准参数（12, 26, 9）。DIF 为快慢 EMA 之差，MACD 柱为 DIF 与 DEA 之差。

- **MACD 收窄**：柱状图绝对值连续缩小，表示当前趋势的动能正在衰减。
- **DIF < 0**：快线位于零轴下方，中期偏空。

### 均线

- **MA20**：20 日简单移动平均线，反映短期趋势。
- **MA60**：60 日简单移动平均线，反映中期趋势。
- **MA20 > MA60**：中期多头排列。

### Pivot Point（枢轴点）

基于前期的高、低、收盘价计算：PP = (High + Low + Close) / 3。

- **P5\_PP**：5 日周期枢轴点，短期支撑/阻力参考。
- **P30\_PP**：30 日周期枢轴点，中期支撑/阻力参考。
- **Close > PP**：价格位于枢轴点之上，偏多。

### 历史波动率

基于对数收益率的滚动标准差，再年化：

```
r_i = ln(Close_i / Close_{i-1})
σ = std(r, window=20) × √252
```

`波动率 < 50%` 表示近期波动处于中低水平。此阈值需根据标的特性校准——对大盘指数偏高，对个股可能偏低。

### |MA20距|（乖离率）

当前价格相对 MA20 的偏离百分比：

```
MA20距 = (Close - MA20) / MA20
```

取绝对值后 ≤ 4.5%，表示价格在均线附近窄幅波动，处于贴线整理状态。

## 各层级解读

### A · 企稳双撑

**条件**：MACD 收窄 + 价格同时站上 5 日和 30 日枢轴点。

**含义**：下跌动能正在衰减，短期和中期支撑均在脚下。这是决策树中信号最强的一档。

**动作**：卖 8% OTM Put。行权价最近，权利金最厚，对后市的信心也最高。

### B1 · 回调均线

**条件**：价格跌破 MA20 但仍在 MA60 之上。

**含义**：中期多头趋势未被破坏，短期出现回调。这是趋势行情中常见的"均线回踩"形态。

**动作**：卖 8% OTM Put。

### B2 · 超跌支撑

**条件**：DIF < 0 但价格仍在 P30\_PP 之上。

**含义**：中期动能偏空，但 30 日枢轴点提供了结构性支撑。弱势中找支撑的逻辑。

**动作**：卖 8% OTM Put。

### B3 · 趋势动能弱

**条件**：MA20 > MA60（多头排列）但 DIF < 0。

**含义**：趋势的骨架还在，但肌肉已经松了。均线多头排列是滞后指标，DIF 转负是领先信号——趋势可能正在转向。

**动作**：卖 10% OTM Put。

### B4 · 低波整理

**条件**：历史波动率 < 50% 且价格贴近 MA20（偏离 ≤ 4.5%）。

**含义**：市场在窄幅震荡，方向不明但波动温和。Theta 衰减对卖方有利，大幅下行的概率偏低。

**动作**：卖 15% OTM Put。低波环境下权利金偏薄，用更深虚值换取安全。

### C1 · 跌势减速

**条件**：Close < MA60 且 |MACD 今| < |MACD 昨|（MACD 收窄）。

**含义**：价格跌破中期均线，但 MACD 柱在缩小，空头力量正在衰减。底部可能正在形成。

**动作**：卖 10% OTM Put。

### C2 · 趋势延续

**条件**：Close ≥ MA20 且 |MA20 偏离| ≤ 10%。不满足 A/B 任何一条。

**含义**：价格在 MA20 上方且偏离合理，趋势仍在运行中，但缺少更强的支撑信号。

**动作**：卖 15% OTM Put。

### C3 · 过热追涨

**条件**：Close ≥ MA20 且 |MA20 偏离| > 10%。

**含义**：价格远超 MA20，短期超买，回调风险增大。

**动作**：卖 15% OTM Put。

### C4 · 加速下杀

**条件**：Close < MA60 且 |MACD 今| ≥ |MACD 昨|（MACD 放大）。

**含义**：价格跌破中期均线且下行动能还在增强，是最危险的市场状态。

**动作**：卖 15% OTM Put。

## OTM 分层逻辑

| 层级 | OTM | 逻辑 |
|------|-----|------|
| A 企稳双撑 | 8% | 信号最强，支撑最可靠 |
| B1 回调均线 | 8% | 均线回踩，中期结构完好 |
| B2 超跌支撑 | 8% | 有 Pivot 支撑兜底 |
| B3 趋势动能弱 | 10% | 趋势尚存但动能衰减，适度保守 |
| B4 低波整理 | 15% | 低波但方向不明，偏保守 |
| C1 跌势减速 | 10% | 空头衰减，底部可能形成，适度保守 |
| C2–C4 | 15% | 兜底层级，更深虚值换安全 |

## 设计备忘

**关于 OTM 分层**：A/B1/B2 用 8%（信号较强，支撑可靠），B3/C1 用 10%（趋势在但动能弱或跌势减速），B4 和 C2-C4 用 15%（偏保守）。参数根据标的杠杆倍数自动缩放。

**关于熔断规则**：基于 TQQQ 回测验证，熔断规则精准命中了 2025 年 2–3 月连续暴跌期的 10 个不安全周（仅 1 个误杀），将执行周平稳到期率从 85.6% 提升至 96.2%。

**关于缺失的退出规则**：Lambda 策略定义了入场条件和行权价选择，但未包含止损、提前平仓和到期移仓的规则。实际部署时需要补充：在什么条件下买回平仓？到期前多少天开始考虑移仓？标的价格跌破行权价后如何应对？

**关于阈值校准**：50% 波动率、4.5% 乖离率、8%/10%/15% OTM 深度——这些参数都是起点而非终点，应根据标的特性和回测结果持续调整。
