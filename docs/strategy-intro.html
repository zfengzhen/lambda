Lambda 策略

动态卖 Put 决策框架

核心思想

Lambda 策略是一个系统化的裸卖看跌期权（Sell Put）决策树。

它不回答"要不要卖 Put"——答案永远是卖。它回答的是：卖多深的虚值？

市场越强，行权价越靠近现价，吃更多权利金；市场越弱，行权价越远离现价，换更大的安全垫。

整套逻辑只用日 K 线数据驱动，不依赖主观判断。

决策树

IF MACD收窄 AND Close > P5_PP AND Close > P30_PP
    → A 企稳双撑 — 卖 8% OTM Put

ELIF Close < MA20 AND Close > MA60
    → B1 回调均线 — 卖 8% OTM Put

ELIF 波动率 < 50% AND |MA20距| ≤ 4.5%
    → B2 低波整理 — 卖 15% OTM Put

ELIF DIF < 0 AND Close > P30_PP
    → B3 超跌支撑 — 卖 8% OTM Put

ELIF MA20 > MA60 AND DIF < 0
    → B4 趋势动能弱 — 卖 10% OTM Put

ELIF Close ≥ MA20 AND |MA20偏离| ≤ 10%
    → C1 趋势延续 — 卖 15% OTM Put

ELIF Close ≥ MA20 AND |MA20偏离| > 10%
    → C2 过热追涨 — 卖 15% OTM Put

ELIF Close < MA60 AND |MACD今| < |MACD昨|
    → C3 跌势减速 — 卖 15% OTM Put

ELSE
    → C4 加速下杀 — 卖 15% OTM Put

优先级自上而下，命中即停。

OTM 分层逻辑

层级OTM逻辑
A 企稳双撑8%信号最强，支撑最可靠
B1 回调均线8%均线回踩，中期结构完好
B2 低波整理15%低波但方向不明，偏保守
B3 超跌支撑8%有 Pivot 支撑兜底
B4 趋势动能弱10%趋势尚存但动能衰减，适度保守
C1 趋势延续15%趋势运行中但缺少强支撑
C2 过热追涨15%短期超买，回调风险增大
C3 跌势减速15%空头衰减，底部可能形成
C4 加速下杀15%最危险状态，下行动能增强

⚠ 连续弱势熔断规则

前 2 周连续 C 类 + 本周也是 C 类时触发判断（本周是 A/B 类始终放行）：

暂停 — 本周 C 类但非 C3：趋势没有减速迹象（C1 还在涨但不稳、C2 过热、C4 加速下杀），风险未收敛
继续卖出 — 本周 C3 + 前 2 周有 C3：跌势已连续收窄（MACD 收窄出现过至少 2 次），空头力量持续衰减，底部信号较可靠
暂停 — 本周 C3 + 前 2 周无 C3：之前都是纯弱势，本周才首次出现减速，可能只是下跌中继的短暂喘息而非真正见底
核心思路：单次 C3 减速不可信，连续减速才可信；A/B 类已有支撑条件，不受熔断影响。

各层级解读

A · 企稳双撑
MACD收窄 AND Close > P5_PP AND Close > P30_PP
下跌动能正在衰减，短期和中期支撑均在脚下。决策树中信号最强的一档。
→ 卖 8% OTM Put — 行权价最近，权利金最厚
B1 · 回调均线
Close < MA20 AND Close > MA60
中期多头趋势未被破坏，短期出现回调。趋势行情中常见的"均线回踩"形态。
→ 卖 8% OTM Put
B2 · 低波整理
波动率 < 50% AND |MA20距| ≤ 4.5%
市场窄幅震荡，方向不明但波动温和。Theta 衰减对卖方有利，大幅下行概率偏低。
→ 卖 15% OTM Put — 低波环境用深虚值换安全
B3 · 超跌支撑
DIF < 0 AND Close > P30_PP
中期动能偏空，但 30 日枢轴点提供结构性支撑。弱势中找支撑的逻辑。
→ 卖 8% OTM Put
B4 · 趋势动能弱
MA20 > MA60 AND DIF < 0
趋势的骨架还在，但肌肉已松。均线多头排列是滞后指标，DIF 转负是领先信号——趋势可能正在转向。
→ 卖 10% OTM Put
C1 · 趋势延续
Close ≥ MA20 AND |MA20偏离| ≤ 10%
价格在 MA20 上方且偏离合理，趋势仍在运行中，但缺少更强的支撑信号。
→ 卖 15% OTM Put
C2 · 过热追涨
Close ≥ MA20 AND |MA20偏离| > 10%
价格远超 MA20，短期超买，回调风险增大。
→ 卖 15% OTM Put
C3 · 跌势减速
Close < MA60 AND |MACD今| < |MACD昨|
价格跌破中期均线，但 MACD 柱在缩小，空头力量正在衰减。底部可能正在形成。
→ 卖 15% OTM Put
C4 · 加速下杀
Close < MA60 AND |MACD今| ≥ |MACD昨|
价格跌破中期均线且下行动能还在增强，是最危险的市场状态。
→ 卖 15% OTM Put

技术指标定义

MACD（12, 26, 9）

$$\text{DIF} = \text{EMA}_{12}(\text{Close}) - \text{EMA}_{26}(\text{Close})$$
$$\text{DEA} = \text{EMA}_{9}(\text{DIF})$$
$$\text{MACD柱} = \text{DIF} - \text{DEA}$$
MACD 收窄$|\text{MACD柱}|$ 连续缩小，表示当前趋势的动能正在衰减
$\text{DIF} < 0$快线位于零轴下方，中期偏空

均线

$$\text{MA}_n = \frac{1}{n}\sum_{i=0}^{n-1} \text{Close}_{t-i}$$
$\text{MA}_{20}$20 日简单移动平均线，反映短期趋势
$\text{MA}_{60}$60 日简单移动平均线，反映中期趋势
$\text{MA}_{20} > \text{MA}_{60}$中期多头排列

Pivot Point（枢轴点）

$$\text{PP} = \frac{\text{High} + \text{Low} + \text{Close}}{3}$$
$\text{P5\_PP}$5 日周期枢轴点，短期支撑/阻力参考
$\text{P30\_PP}$30 日周期枢轴点，中期支撑/阻力参考
$\text{Close} > \text{PP}$价格位于枢轴点之上，偏多

历史波动率

基于对数收益率的滚动标准差，再年化：

$$r_i = \ln\!\left(\frac{\text{Close}_i}{\text{Close}_{i-1}}\right)$$
$$\sigma = \text{std}(r,\; \text{window}=20) \times \sqrt{252}$$

$\sigma < 50\%$ 表示近期波动处于中低水平。此阈值需根据标的特性校准——对大盘指数偏高，对个股可能偏低。

|MA20距|（乖离率）

$$\text{BIAS}_{20} = \frac{\text{Close} - \text{MA}_{20}}{\text{MA}_{20}} \times 100\%$$

$|\text{BIAS}_{20}| \le 4.5\%$，表示价格在均线附近窄幅波动，处于贴线整理状态。

设计备忘

关于 OTM 分层
A/B1/B3 用 8%（信号较强，支撑可靠），B4 用 10%（趋势在但动能弱），B2 和 C 类用 15%（偏保守）。参数根据标的杠杆倍数自动缩放。
关于熔断规则
基于 TQQQ 回测验证，熔断规则精准命中了 2025 年 2–3 月连续暴跌期的 10 个不安全周（仅 1 个误杀），将执行周平稳到期率从 85.6% 提升至 96.2%。
关于缺失的退出规则
Lambda 策略定义了入场条件和行权价选择，但未包含止损、提前平仓和到期移仓的规则。实际部署时需要补充：在什么条件下买回平仓？到期前多少天开始考虑移仓？标的价格跌破行权价后如何应对？
关于阈值校准
50% 波动率、4.5% 乖离率、8%/10%/15% OTM 深度——这些参数都是起点而非终点，应根据标的特性和回测结果持续调整。
